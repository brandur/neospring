neospring

This is an installation of Neospring, an implementation of the Spring '83 protocol.

If this is neospring.brandur.org, then it's run by Brandur (brandur@ my domain) and hosts my board, which you can request like so: 

    
        $ curl https://neospring.brandur.org/2c98169d0b6fa73cab5a830be8dde53c5f388d5c6f8e6f756b6b6dbcc83e1124

Board TTL is configured to be {{.BoardTTLDays}} days, as recommended by the specification.

Server code is written to be highly available and for robustness, and has a comprehensive test suite, but so far has little real-world vetting so far, so use with caution.
